# blockudoku
A hybrid connection of Tetris and Sudoku. Players must fit blocks into a sudoku grid, earning points by completing rows,columns, or a 3x3 subgrid. Perhaps it's more like a block puzzle than tetris, but whatever, Blockudoku sounds cool.

## How to Play
Simply drag a shape into the grid and fill as many rows, columns, and 3x3 grids as you can to earn points. The game ends when a shape can no longer fit into the grid.
